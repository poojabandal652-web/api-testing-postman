# 🛒 E-Commerce API Testing Project

## 📌 Project Overview

This project demonstrates API testing using Postman for an E-Commerce application. It includes validation of multiple APIs such as login, product retrieval, and user profile.

## 🛠️ Tools Used

* Postman
* JavaScript (Postman scripting)
* Git & GitHub

## 🔍 Test Scenarios Covered

* ✅ Status code validation
* ✅ Response body validation
* ✅ JSON parsing using `pm.response.json()`
* ✅ Dynamic data handling
* ✅ Environment variables usage
* ✅ API chaining (product_id extraction)

## 🔗 APIs Tested

* Login API (POST)
* Get Products (GET)
* Get User Profile (GET)

## ▶️ How to Run

1. Import collection into Postman
2. Import environment file
3. Select environment
4. Run using Collection Runner

## 💡 Key Learning

* Extracted values using `pm.response.json()`
* Stored variables using `pm.environment.set()`
* Handled dynamic API data
